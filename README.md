# Buy Market Exchange - Backend 🛒♻️

Welcome to the backend repository of **Buy Market Exchange**, a comprehensive multi-vendor marketplace and product exchange platform integrated with AI-driven analysis services. 

This enterprise-grade backend is built from scratch using **NestJS** and **TypeScript**, enforcing strict modular architecture, clean code practices, and scalable design patterns.

---

## 🚀 Key Features & Architectural Highlights

**Modular Architecture:** Built using NestJS modules to ensure high maintainability, separation of concerns, and loose coupling between features (Users, Products, Categories, Reports, Market Data).
  
**Role-Based Access Control (RBAC):** Secure authentication and authorization flow using **JWT (JSON Web Tokens)**, managing different access levels for *Admins*, *Vendors*, and *Customers*.

 **AI-Driven Product Analysis & Market Data:** Integrated third-party AI services to analyze product data against live market trends, providing automated metrics and insights stored within `Market Data`.
 
 **Robust Admin Dashboard & Reporting:

Engineered optimized RESTful endpoints tailored to power a data-heavy management dashboard with complex filtering, handling systemic tracking through `Reports`.
 
 **Data Validation & DTOs:** Strict request payload validation using `class-validator` and `class-transformer` to 
 
 secure endpoints against malformed data.

---

## 🛠️ Tech Stack & Tools

*   **Framework:** NestJS (Node.js)
*   **Language:** TypeScript
*   **Database:** MongoDB
*   **ODM:** Mongoose / `@nestjs/mongoose`
*   **Authentication:** Passport.js & JWT
*   **API Documentation:** Swagger UI (OpenAPI)

---

## 📋 Database Schema & Models

The MongoDB database is designed to handle high-performance document storage with optimized indexing and relations managed at the application layer:

*   **User Model:** Manages profile segregation, credentials, and Role-Based Authorization (Admin, Vendor, Customer).
*   **Product Model:** Handles the core marketplace catalog, vendor ownership, pricing, and specifications for exchange.
*   
*   **Category Model:** Provides dynamic multi-level categorization for seamless product discovery and filtering.
*   
*   **Report Model:** Tracks system compliance, user flags, and automated logs generated by backend analysis.
  
*  **Market Data Model:** Stores aggregated data and insights from the AI product analysis engine to track platform exchange metrics.
